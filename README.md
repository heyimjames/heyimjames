I run [OCTOBER](https://octoberwip.com), an independent design studio for startups and VCs. We ship products people actually want to use.

**11 years in Product Design, Brand and Marketing.** I build web apps, React Native apps, native SwiftUI for iOS and Mac, Chrome extensions, landing pages and funnels. Through the studio, I've led UI/UX and brand for VC funds, AI startups, fintech, beauty tech, consumer apps and B2B SaaS.
